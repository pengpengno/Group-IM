## 🚀 快速开始

### 新手推荐阅读顺序
1. **阿里云部署指南** - 了解整体部署架构和环境准备
2. **CI/CD 标准流程** - 理解自动化部署流程
3. **宝塔面板配置手册** - 学习日常运维操作

### 场景化阅读

#### 首次部署
```
阿里云部署指南 → 环境准备
     ↓
阿里云部署指南 → 项目部署
     ↓
宝塔面板配置手册 → Docker 管理器
     ↓
CI/CD 标准流程 → 配置自动部署
```

#### 日常运维
```
宝塔面板配置手册 → 监控与告警
     ↓
宝塔面板配置手册 → 日志管理
     ↓
宝塔面板配置手册 → 备份策略
```

#### 故障排查
```
阿里云部署指南 → 故障排查章节
     ↓
宝塔面板配置手册 → 常见问题
     ↓
CI/CD 标准流程 → 回滚策略
```
